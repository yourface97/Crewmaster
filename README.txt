This is a sub-application for a production planning scheduling application. It is used to maintain a master list of associates for the main scheduling application. This is a list of the features this sub-application intends to provide (some or all of these features may not yet be functional): 
-Filter through master list using the Business Unit, Area, and Crew drop down selectors. 
-Add associate to the master list by completing the entire form and submitting. 
-Remove associate from the master list by clicking delete button next to their table entry.
